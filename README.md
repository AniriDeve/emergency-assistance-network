# Emergency Assistance Network

##  Обзор проекта
Emergency Assistance Network — backend-сервис для экстренной помощи людям с инвалидностью и маломобильным гражданам.
Это социально значимый проект, направленный на повышение безопасности людей с инвалидностью.

Сервис позволяет пользователю отправить тревожный сигнал в один клик, автоматически зафиксировать обращение,
передать его операторам и сохранить всю информацию для последующего анализа и контроля.

---

## Значимость проекта
Проект разработан по заказу социального фонда совместно с экстренными и социальными службами города.

Основная задача — обеспечить надёжный и доступный канал связи для людей,
которые не всегда могут воспользоваться телефонным звонком в экстренной ситуации.

---

##  Цель проекта
- сокращение времени реагирования
- повышение безопасности людей с инвалидностью
- централизованный учёт экстренных обращений
- прозрачность работы экстренных служб

---

##  Особенности проекта
- создание экстренного обращения
- хранение геолокации пользователя
- указание типа необходимой помощи
- хранение медицинской информации
- управление статусами обращения
- журналирование событий
